# Hópverkefni 2


### Til að keyra 

Til að setja up tól: \
npm install 

Til að athuga lint: \
npm run test 

Til að keyra á localhost: \
npm run dev 

### Lýsing á upsetningu
styles.scss í ./src/styles möppu \
javascript í ./src/lib möppu 

við keyrslu á npm run dev er styles.scss transpiluð í styles.css og scripts.js transpiluð og böndluð yfir í dist möppu \
